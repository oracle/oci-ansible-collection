.. Title

oracle.oci.oci_governance_rules_control_plane_governance_rule -- Manage a GovernanceRule resource in Oracle Cloud Infrastructure
++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++

.. Collection note

.. note::
    This plugin is part of the `oracle.oci collection <https://galaxy.ansible.com/oracle/oci>`_ (version 4.11.0).

    You might already have this collection installed if you are using the ``ansible`` package.
    It is not included in ``ansible-core``.
    To check whether it is installed, run :code:`ansible-galaxy collection list`.

    To install it, use: :code:`ansible-galaxy collection install oracle.oci`.

    To use it in a playbook, specify: :code:`oracle.oci.oci_governance_rules_control_plane_governance_rule`.

.. versionadded:: 2.9.0 of oracle.oci

.. contents::
   :local:
   :depth: 1

.. Deprecated


Synopsis
--------

.. Description

- This module allows the user to create, update and delete a GovernanceRule resource in Oracle Cloud Infrastructure
- For *state=present*, create governance rule in the root compartment only. Either relatedResourceId or template must be supplied.
- This resource has the following action operations in the :ref:`oracle.oci.oci_governance_rules_control_plane_governance_rule_actions <ansible_collections.oracle.oci.oci_governance_rules_control_plane_governance_rule_actions_module>` module: retry.


.. Aliases


.. Requirements

Requirements
------------
The below requirements are needed on the host that executes this module.

- python >= 3.6
- Python SDK for Oracle Cloud Infrastructure https://oracle-cloud-infrastructure-python-sdk.readthedocs.io
