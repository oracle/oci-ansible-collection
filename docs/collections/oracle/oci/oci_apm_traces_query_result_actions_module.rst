oracle.oci.oci_apm_traces_query_result_actions -- Perform actions on a QueryResult resource in Oracle Cloud Infrastructure
++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++

.. Collection note

.. note::

Synopsis
--------

.. Description

- Perform actions on a QueryResult resource in Oracle Cloud Infrastructure
- For *action=query*, retrieves the results (selected attributes and aggregations) of a query constructed according to the Application Performance Monitoring Defined Query Syntax. Query results are filtered by the filter criteria specified in the where clause. Further query results are grouped by the attributes specified in the group by clause.  Finally, ordering (asc/desc) is done by the specified attributes in the order by clause.
