.. Title

oracle.oci.oci_service_catalog_association -- Manage a ServiceCatalogAssociation resource in Oracle Cloud Infrastructure
++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++

.. Collection note

.. note::
    This plugin is part of the `oracle.oci collection <https://galaxy.ansible.com/oracle/oci>`_ (version 2.30.0).

    To install it use: :code:`ansible-galaxy collection install oracle.oci`.

    To use it in a playbook, specify: :code:`oracle.oci.oci_service_catalog_association`.

.. version_added

.. versionadded:: 2.9 of oracle.oci

.. contents::
   :local:
   :depth: 1

.. Deprecated


Synopsis
--------

.. Description

- This module allows the user to create and delete a ServiceCatalogAssociation resource in Oracle Cloud Infrastructure
- For *state=present*, creates an association between service catalog and a resource.


.. Aliases


.. Requirements

Requirements
------------
The below requirements are needed on the host that executes this module.

- python >= 3.6
- Python SDK for Oracle Cloud Infrastructure https://oracle-cloud-infrastructure-python-sdk.readthedocs.io

.. Examples

Examples
--------

.. code-block:: yaml+jinja

    
    - name: Create service_catalog_association
      oci_service_catalog_association:
        service_catalog_id: "ocid1.servicecatalog.oc1..xxxxxxEXAMPLExxxxxx"
        entity_id: "ocid1.entity.oc1..xxxxxxEXAMPLExxxxxx"

    - name: Delete service_catalog_association
      oci_service_catalog_association:
        service_catalog_association_id: "ocid1.servicecatalogassociation.oc1..xxxxxxEXAMPLExxxxxx"
        state: absent
