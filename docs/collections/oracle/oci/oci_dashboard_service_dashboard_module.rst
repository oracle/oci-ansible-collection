.. Title

oracle.oci.oci_dashboard_service_dashboard -- Manage a Dashboard resource in Oracle Cloud Infrastructure
++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++

.. Collection note

.. note::
    This plugin is part of the `oracle.oci collection <https://galaxy.ansible.com/oracle/oci>`_ (version 4.34.0).

    You might already have this collection installed if you are using the ``ansible`` package.
    It is not included in ``ansible-core``.
    To check whether it is installed, run :code:`ansible-galaxy collection list`.

    To install it, use: :code:`ansible-galaxy collection install oracle.oci`.

    To use it in a playbook, specify: :code:`oracle.oci.oci_dashboard_service_dashboard`.

.. version_added

.. versionadded:: 2.9.0 of oracle.oci

.. contents::
   :local:
   :depth: 1

.. Deprecated


Synopsis
--------

.. Description

- This module allows the user to create, update and delete a Dashboard resource in Oracle Cloud Infrastructure
- For *state=present*, creates a new dashboard in the dashboard group's compartment using the details provided in request body.
- **Caution:** Resources for the Dashboard service are created in the tenacy's home region. Although it is possible to create dashboard resource in regions other than the home region, you won't be able to view those resources in the Console. Therefore, creating resources outside of the home region is not recommended.
- This resource has the following action operations in the :ref:`oracle.oci.oci_dashboard_service_dashboard_actions <ansible_collections.oracle.oci.oci_dashboard_service_dashboard_actions_module>` module: change_dashboard_group.


.. Aliases


.. Requirements

Requirements
------------
The below requirements are needed on the host that executes this module.

- python >= 3.6
- Python SDK for Oracle Cloud Infrastructure https://oracle-cloud-infrastructure-python-sdk.readthedocs.io

.. Examples

Examples
--------

.. code-block:: yaml+jinja

    
    - name: Create dashboard with schema_version = V1
      oci_dashboard_service_dashboard:
        # required
        dashboard_group_id: "ocid1.dashboardgroup.oc1..xxxxxxEXAMPLExxxxxx"
        schema_version: V1

        # optional
        display_name: display_name_example
        description: description_example
        freeform_tags: {'Department': 'Finance'}
        defined_tags: {'Operations': {'CostCenter': 'US'}}
        config: null
        widgets: [ "widgets_example" ]

    - name: Update dashboard with schema_version = V1
      oci_dashboard_service_dashboard:
        # required
        schema_version: V1

        # optional
        display_name: display_name_example
        description: description_example
        freeform_tags: {'Department': 'Finance'}
        defined_tags: {'Operations': {'CostCenter': 'US'}}
        config: null
        widgets: [ "widgets_example" ]

    - name: Update dashboard using name (when environment variable OCI_USE_NAME_AS_IDENTIFIER is set) with schema_version = V1
      oci_dashboard_service_dashboard:
        # required
        dashboard_group_id: "ocid1.dashboardgroup.oc1..xxxxxxEXAMPLExxxxxx"
        schema_version: V1

        # optional
        display_name: display_name_example
        description: description_example
        freeform_tags: {'Department': 'Finance'}
        defined_tags: {'Operations': {'CostCenter': 'US'}}
        config: null
        widgets: [ "widgets_example" ]

    - name: Delete dashboard
      oci_dashboard_service_dashboard:
        # required
        dashboard_id: "ocid1.dashboard.oc1..xxxxxxEXAMPLExxxxxx"
        state: absent

        # optional
        opc_cross_region: us-phoenix-1

    - name: Delete dashboard using name (when environment variable OCI_USE_NAME_AS_IDENTIFIER is set)
      oci_dashboard_service_dashboard:
        # required
        dashboard_group_id: "ocid1.dashboardgroup.oc1..xxxxxxEXAMPLExxxxxx"
        display_name: display_name_example
        state: absent
