.. Title

oracle.oci.oci_apm_control_plane_data_keys_actions -- Perform actions on a DataKeys resource in Oracle Cloud Infrastructure
+++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++

.. Collection note

.. note::
    This plugin is part of the `oracle.oci collection <https://galaxy.ansible.com/oracle/oci>`_ (version 2.52.0).

    You might already have this collection installed if you are using the ``ansible`` package.

    To check whether it is installed, run :code:`ansible-galaxy collection list`.

.. Deprecated


Synopsis
--------

.. Description

- Perform actions on a DataKeys resource in Oracle Cloud Infrastructure
- For *action=generate*, generates a set of new Data Keys for the specified APM domain with the specified names and types. These will be added to the existing set of Data Keys for the specified APM domain.
- For *action=remove*, removes the set of specified Data Keys from the specified APM domain. Agents would no longer be able to use these data keys to upload to the APM domain once this operation is completed.
