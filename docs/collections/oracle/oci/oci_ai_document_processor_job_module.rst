.. Title

oracle.oci.oci_ai_document_processor_job -- Manage a ProcessorJob resource in Oracle Cloud Infrastructure
+++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++

.. Collection note

.. note::
    This plugin is part of the `oracle.oci collection <https://galaxy.ansible.com/oracle/oci>`_ (version 4.17.0).

    You might already have this collection installed if you are using the ``ansible`` package.
    It is not included in ``ansible-core``.
    To check whether it is installed, run :code:`ansible-galaxy collection list`.

    To install it, use: :code:`ansible-galaxy collection install oracle.oci`.

    To use it in a playbook, specify: :code:`oracle.oci.oci_ai_document_processor_job`.

.. version_added

.. versionadded:: 2.9.0 of oracle.oci

.. contents::
   :local:
   :depth: 1

.. Deprecated


Synopsis
--------

.. Description

- This module allows the user to create a ProcessorJob resource in Oracle Cloud Infrastructure
- For *state=present*, create a processor job for document analysis.
- This resource has the following action operations in the :ref:`oracle.oci.oci_ai_document_processor_job_actions <ansible_collections.oracle.oci.oci_ai_document_processor_job_actions_module>` module: cancel.


.. Aliases


.. Requirements

Requirements
------------
The below requirements are needed on the host that executes this module.

- python >= 3.6
- Python SDK for Oracle Cloud Infrastructure https://oracle-cloud-infrastructure-python-sdk.readthedocs.io

.. Examples

Examples
--------

.. code-block:: yaml+jinja

    
    - name: Create processor_job
      oci_ai_document_processor_job:
        # required
        input_location:
          # required
          data: data_example
          source_type: INLINE_DOCUMENT_CONTENT
        output_location:
          # required
          namespace_name: namespace_name_example
          bucket_name: bucket_name_example
          prefix: prefix_example
        compartment_id: "ocid1.compartment.oc1..xxxxxxEXAMPLExxxxxx"
        processor_config:
          # required
          processor_type: GENERAL
          features:
          - # required
            feature_type: DOCUMENT_CLASSIFICATION

            # optional
            max_results: 56

            # optional
          document_type: INVOICE
          is_zip_output_enabled: true
          language: language_example

        # optional
        display_name: display_name_example
