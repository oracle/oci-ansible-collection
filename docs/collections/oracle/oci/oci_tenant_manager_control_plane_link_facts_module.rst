.. Title

oracle.oci.oci_tenant_manager_control_plane_link_facts -- Fetches details about one or multiple Link resources in Oracle Cloud Infrastructure
+++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++

.. Collection note

.. note::
    This plugin is part of the `oracle.oci collection <https://galaxy.ansible.com/oracle/oci>`_ (version 4.34.0).

    You might already have this collection installed if you are using the ``ansible`` package.
    It is not included in ``ansible-core``.
    To check whether it is installed, run :code:`ansible-galaxy collection list`.

    To install it, use: :code:`ansible-galaxy collection install oracle.oci`.

    To use it in a playbook, specify: :code:`oracle.oci.oci_tenant_manager_control_plane_link_facts`.

.. version_added

.. versionadded:: 2.9.0 of oracle.oci

.. contents::
   :local:
   :depth: 1

.. Deprecated


Synopsis
--------

.. Description

- Fetches details about one or multiple Link resources in Oracle Cloud Infrastructure
- Return a (paginated) list of links.
- If *link_id* is specified, the details of a single Link will be returned.


.. Aliases


.. Requirements

Requirements
------------
The below requirements are needed on the host that executes this module.

- python >= 3.6
- Python SDK for Oracle Cloud Infrastructure https://oracle-cloud-infrastructure-python-sdk.readthedocs.io

.. Examples

Examples
--------

.. code-block:: yaml+jinja

    
    - name: Get a specific link
      oci_tenant_manager_control_plane_link_facts:
        # required
        link_id: "ocid1.link.oc1..xxxxxxEXAMPLExxxxxx"

    - name: List links
      oci_tenant_manager_control_plane_link_facts:

        # optional
        parent_tenancy_id: "ocid1.parenttenancy.oc1..xxxxxxEXAMPLExxxxxx"
        child_tenancy_id: "ocid1.childtenancy.oc1..xxxxxxEXAMPLExxxxxx"
        lifecycle_state: CREATING
        sort_order: ASC
