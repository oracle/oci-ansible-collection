.. Title

oracle.oci.oci_resource_manager_job_facts -- Fetches details about one or multiple Job resources in Oracle Cloud Infrastructure
+++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++

.. Collection note

.. note::
    This plugin is part of the `oracle.oci collection <https://galaxy.ansible.com/oracle/oci>`_ (version 4.34.0).

    You might already have this collection installed if you are using the ``ansible`` package.

    To check whether it is installed, run :code:`ansible-galaxy collection list`.

Synopsis
--------

.. Description

- Fetches details about one or multiple Job resources in Oracle Cloud Infrastructure
- Lists jobs according to the specified filter. By default, the list is ordered by time created.
- - To list all jobs in a stack, provide the stack `OCID <https://docs.cloud.oracle.com/iaas/Content/General/Concepts/identifiers.htm>`_. - To list all jobs in a compartment, provide the compartment `OCID <https://docs.cloud.oracle.com/iaas/Content/General/Concepts/identifiers.htm>`_. - To return a specific job, provide the job `OCID <https://docs.cloud.oracle.com/iaas/Content/General/Concepts/identifiers.htm>`_. (Equivalent to `GetStack <https://docs.cloud.oracle.com/en-us/iaas/api/#/en/resourcemanager/latest/Stack/GetStack>`_.)
- If *job_id* is specified, the details of a single Job will be returned.
