.. Title

oracle.oci.oci_blockstorage_volume_facts -- Fetches details about one or multiple Volume resources in Oracle Cloud Infrastructure
+++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++

.. Collection note

.. note::
    This plugin is part of the `oracle.oci collection <https://galaxy.ansible.com/oracle/oci>`_ (version 4.2.0).

    You might already have this collection installed if you are using the ``ansible`` package.
    It is not included in ``ansible-core``.
    To check whether it is installed, run :code:`ansible-galaxy collection list`.

    To install it, use: :code:`ansible-galaxy collection install oracle.oci`.

    To use it in a playbook, specify: :code:`oracle.oci.oci_blockstorage_volume_facts`.

.. version_added

.. versionadded:: 2.9.0 of oracle.oci

.. contents::
   :local:
   :depth: 1

.. Deprecated


Synopsis
--------

.. Description

- Fetches details about one or multiple Volume resources in Oracle Cloud Infrastructure
- Lists the volumes in the specified compartment and availability domain.
- If *volume_id* is specified, the details of a single Volume will be returned.


.. Aliases


.. Requirements

Requirements
------------
The below requirements are needed on the host that executes this module.

- python >= 3.6
- Python SDK for Oracle Cloud Infrastructure https://oracle-cloud-infrastructure-python-sdk.readthedocs.io

.. Examples

Examples
--------

.. code-block:: yaml+jinja

    
    - name: Get a specific volume
      oci_blockstorage_volume_facts:
        # required
        volume_id: "ocid1.volume.oc1..xxxxxxEXAMPLExxxxxx"

    - name: List volumes
      oci_blockstorage_volume_facts:

        # optional
        availability_domain: Uocm:PHX-AD-1
        compartment_id: "ocid1.compartment.oc1..xxxxxxEXAMPLExxxxxx"
        display_name: display_name_example
        sort_by: TIMECREATED
        sort_order: ASC
        volume_group_id: "ocid1.volumegroup.oc1..xxxxxxEXAMPLExxxxxx"
        lifecycle_state: PROVISIONING
