.. Title

oracle.oci.oci_functions_function_actions -- Perform actions on a Function resource in Oracle Cloud Infrastructure
++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++

.. Collection note

.. note::
    This plugin is part of the `oracle.oci collection <https://galaxy.ansible.com/oracle/oci>`_.

    To install it use: :code:`ansible-galaxy collection install oracle.oci`.

    To use it in a playbook, specify: :code:`oracle.oci.oci_functions_function_actions`.

.. version_added

.. versionadded:: 2.9 of oracle.oci

.. contents::
   :local:
   :depth: 1

.. Deprecated


Synopsis
--------

.. Description

- Perform actions on a Function resource in Oracle Cloud Infrastructure
- For *action=invoke*, invokes a function

.. Aliases


.. Requirements

Requirements
------------
The below requirements are needed on the host that executes this module.

- python >= 2.7
- Python SDK for Oracle Cloud Infrastructure https://oracle-cloud-infrastructure-python-sdk.readthedocs.io

Examples

.. note::
    These examples assume the ``collections`` keyword is defined in  playbook and do not use the fully qualified collection name.

    - name: Perform action invoke on function
      oci_functions_function_actions:
        function_id: ocid1.function.oc1..xxxxxxEXAMPLExxxxxx
        action: invoke
