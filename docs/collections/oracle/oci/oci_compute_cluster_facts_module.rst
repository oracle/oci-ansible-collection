.. Title

oracle.oci.oci_compute_cluster_facts -- Fetches details about one or multiple ComputeCluster resources in Oracle Cloud Infrastructure
+++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++

.. Collection note

.. note::
    This plugin is part of the `oracle.oci collection <https://galaxy.ansible.com/oracle/oci>`_ (version 4.34.0).

    You might already have this collection installed if you are using the ``ansible`` package.
    It is not included in ``ansible-core``.
    To check whether it is installed, run :code:`ansible-galaxy collection list`.

    To install it, use: :code:`ansible-galaxy collection install oracle.oci`.

    To use it in a playbook, specify: :code:`oracle.oci.oci_compute_cluster_facts`.

.. version_added

.. versionadded:: 2.9.0 of oracle.oci

.. contents::
   :local:
   :depth: 1

.. Deprecated


Synopsis
--------

.. Description

- Fetches details about one or multiple ComputeCluster resources in Oracle Cloud Infrastructure
- Lists the compute clusters in the specified compartment. A `compute cluster <https://docs.cloud.oracle.com/iaas/Content/Compute/Tasks/compute-clusters.htm>`_ is a remote direct memory access (RDMA) network group.
- If *compute_cluster_id* is specified, the details of a single ComputeCluster will be returned.


.. Aliases


.. Requirements

Requirements
------------
The below requirements are needed on the host that executes this module.

- python >= 3.6
- Python SDK for Oracle Cloud Infrastructure https://oracle-cloud-infrastructure-python-sdk.readthedocs.io

.. Examples

Examples
--------

.. code-block:: yaml+jinja

    
    - name: Get a specific compute_cluster
      oci_compute_cluster_facts:
        # required
        compute_cluster_id: "ocid1.computecluster.oc1..xxxxxxEXAMPLExxxxxx"

    - name: List compute_clusters
      oci_compute_cluster_facts:
        # required
        compartment_id: "ocid1.compartment.oc1..xxxxxxEXAMPLExxxxxx"

        # optional
        availability_domain: Uocm:PHX-AD-1
        display_name: display_name_example
        sort_by: TIMECREATED
        sort_order: ASC
