.. Title

oracle.oci.oci_dts_appliance_export_job_facts -- Fetches details about one or multiple ApplianceExportJob resources in Oracle Cloud Infrastructure
++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++

.. Collection note

.. note::
    This plugin is part of the `oracle.oci collection <https://galaxy.ansible.com/oracle/oci>`_ (version 4.34.0).

    You might already have this collection installed if you are using the ``ansible`` package.
    It is not included in ``ansible-core``.
    To check whether it is installed, run :code:`ansible-galaxy collection list`.

    To install it, use: :code:`ansible-galaxy collection install oracle.oci`.

    To use it in a playbook, specify: :code:`oracle.oci.oci_dts_appliance_export_job_facts`.

.. version_added

.. versionadded:: 2.9.0 of oracle.oci

.. contents::
   :local:
   :depth: 1

.. Deprecated


Synopsis
--------

.. Description

- Fetches details about one or multiple ApplianceExportJob resources in Oracle Cloud Infrastructure
- Lists Appliance Export Jobs in a given compartment
- If *appliance_export_job_id* is specified, the details of a single ApplianceExportJob will be returned.


.. Aliases


.. Requirements

Requirements
------------
The below requirements are needed on the host that executes this module.

- python >= 3.6
- Python SDK for Oracle Cloud Infrastructure https://oracle-cloud-infrastructure-python-sdk.readthedocs.io

Examples
--------

.. code-block:: yaml+jinja

    
    - name: Get a specific appliance_export_job
      oci_dts_appliance_export_job_facts:
        # required
        appliance_export_job_id: "ocid1.applianceexportjob.oc1..xxxxxxEXAMPLExxxxxx"

    - name: List appliance_export_jobs
      oci_dts_appliance_export_job_facts:
        # required
        compartment_id: "ocid1.compartment.oc1..xxxxxxEXAMPLExxxxxx"

        # optional
        lifecycle_state: CREATING
        display_name: display_name_example
