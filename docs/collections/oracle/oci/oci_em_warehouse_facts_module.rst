.. Title

oracle.oci.oci_em_warehouse_facts -- Fetches details about one or multiple EmWarehouse resources in Oracle Cloud Infrastructure
+++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++

.. Collection note

.. note::
    This plugin is part of the `oracle.oci collection <https://galaxy.ansible.com/oracle/oci>`_ (version 5.5.0).

    You might already have this collection installed if you are using the ``ansible`` package.
    It is not included in ``ansible-core``.
    To check whether it is installed, run :code:`ansible-galaxy collection list`.

    To install it, use: :code:`ansible-galaxy collection install oracle.oci`.

    To use it in a playbook, specify: :code:`oracle.oci.oci_em_warehouse_facts`.

.. version_added

.. versionadded:: 2.9.0 of oracle.oci

.. contents::
   :local:
   :depth: 1

.. Deprecated


Synopsis
--------

.. Description

- Fetches details about one or multiple EmWarehouse resources in Oracle Cloud Infrastructure
- Returns a list of EmWarehouses.
- If *em_warehouse_id* is specified, the details of a single EmWarehouse will be returned.


.. Aliases


.. Requirements

Requirements
------------
The below requirements are needed on the host that executes this module.

- python >= 3.6
- Python SDK for Oracle Cloud Infrastructure https://oracle-cloud-infrastructure-python-sdk.readthedocs.io

Examples
--------

.. code-block:: yaml+jinja

    
    - name: Get a specific em_warehouse
      oci_em_warehouse_facts:
        # required
        em_warehouse_id: "ocid1.emwarehouse.oc1..xxxxxxEXAMPLExxxxxx"

    - name: List em_warehouses
      oci_em_warehouse_facts:

        # optional
        compartment_id: "ocid1.compartment.oc1..xxxxxxEXAMPLExxxxxx"
        lifecycle_state: CREATING
        display_name: display_name_example
        operations_insights_warehouse_id: "ocid1.operationsinsightswarehouse.oc1..xxxxxxEXAMPLExxxxxx"
        sort_order: ASC
        sort_by: timeCreated
