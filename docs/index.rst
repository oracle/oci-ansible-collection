Oracle Cloud Infrastructure Ansible Modules Collection - Modules Index
===========================================================================================
`Oracle Cloud Infrastructure Ansible Modules Collection <https://github.com/oracle/oci-ansible-collection>`_ provide an easy way to create and provision resources in Oracle Cloud Infrastructure (OCI) through Ansible. These modules allow you to author Ansible playbooks that help you automate the provisioning and configuring of Oracle Cloud Infrastructure services and resources, such as Compute, Load Balancing, Database, and other Oracle Cloud Infrastructure services.

- Documentation and getting started guide: https://docs.cloud.oracle.com/iaas/Content/API/SDKDocs/ansible.htm
- Github Project URL: https://github.com/oracle/oci-ansible-collection
