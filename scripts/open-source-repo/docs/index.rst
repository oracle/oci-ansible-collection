Oracle Cloud Infrastructure Ansible Modules
===========================================================================================
`Oracle Cloud Infrastructure Ansible Modules <https://github.com/oracle/oci-ansible-modules>`_ provide an easy way to create and provision resources in Oracle Cloud Infrastructure (OCI) through Ansible. These modules allow you to author Ansible playbooks that help you automate the provisioning and configuring of Oracle Cloud Infrastructure services and resources, such as Compute, Load Balancing, Database, and other Oracle Cloud Infrastructure services.
